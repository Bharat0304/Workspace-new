# Extension Icons

This directory contains the browser extension icons in various sizes.

## Required Icons

- **icon16.png** - 16x16 pixels (toolbar, favicon)
- **icon48.png** - 48x48 pixels (extension management)
- **icon128.png** - 128x128 pixels (web store, installation)

## Icon Design

The icons should represent:
- 🎯 Focus and productivity
- 🧠 AI intelligence
- 📚 Educational focus
- 🚫 Distraction blocking
- 🌟 Achievement and motivation

## Temporary Solution

For development, you can use simple colored squares or download free icons from:
- https://www.flaticon.com/
- https://feathericons.com/
- https://heroicons.com/

## Recommended Icon Style

- **Primary Color**: Blue/Purple gradient (#6366f1 to #8b5cf6)
- **Design**: Clean, modern, minimalist
- **Symbol**: Brain + Target + Shield combination
- **Background**: Rounded square or circle

## Adding Icons

1. Create or download PNG files in the required sizes
2. Save them as:
   - `icon16.png`
   - `icon48.png` 
   - `icon128.png`
3. Reload the extension in Chrome

## Quick Fix

For immediate testing, you can:
1. Create simple colored squares in any image editor
2. Use different colors for each size to verify loading
3. Replace with proper icons later
